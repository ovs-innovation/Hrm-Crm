# Trial Experience — Vastora CRM

**Version:** 1.0.0-draft  
**Duration:** 14 days on Growth tier

---

## Trial Provisions

| Feature | Trial Access |
|---------|--------------|
| Modules | Full Growth (CRM + HRM + Projects) |
| Users | Max 10 |
| Leads / Deals | Full limits |
| AI | 50 total requests |
| API | Disabled |
| Custom domain | Disabled |
| "Powered by Vastora" on PDFs | Yes |
| Support | Email only, 48h SLA |

---

## Trial Email Sequence

| Day | Email | CTA |
|-----|-------|-----|
| 0 | Welcome + getting started guide | Complete setup wizard |
| 1 | "Create your first lead in 2 minutes" | Video + link |
| 3 | Feature highlight: Pipeline board | Open deals board |
| 7 | Case study: How Agency X closed 30% more deals | Read story |
| 10 | "Your team can join free during trial" | Invite team |
| 12 | "Trial ends in 2 days" | Upgrade now |
| 14 | Trial expired — read-only mode | Upgrade to restore |
| 17 | Account suspended notice | Upgrade within 30 days |

---

## In-App Trial Banner

```
┌─────────────────────────────────────────────────────────────┐
│ ⏱ 12 days left in your trial. [Upgrade Now]  [Dismiss]     │
└─────────────────────────────────────────────────────────────┘
```

Shown on all pages; dismissible per session only.

---

## Post-Trial (Grace Period — 3 Days)

- Read-only access: view data, no create/edit
- Export allowed (CSV)
- Upgrade restores full access instantly

---

## Conversion Flow

Upgrade button → Plan selection → Payment (Razorpay) → Confirmation → Remove trial banner + unlock limits

---

## Demo Mode (Sales-Assisted Trials)

Sales can extend trial to 30 days via admin coupon `TRIAL30`.  
Sales can pre-seed demo data pack before prospect login.
