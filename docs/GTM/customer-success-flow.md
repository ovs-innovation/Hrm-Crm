# Customer Success Flow — Vastora CRM

**Version:** 1.0.0-draft

---

## Lifecycle Stages

```
Trial → Activation → Adoption → Value → Expansion → Renewal
```

---

## Stage 1 — Trial (Day 0–14)

| Action | Owner | Trigger |
|--------|-------|---------|
| Welcome email + setup guide | Automated | Signup |
| Check wizard completion | CS (Enterprise) / Automated | Day 1 |
| CRM onboarding checklist | In-app | Day 0 |
| Trial reminder emails | Automated | Day 12, 14 |
| Offer demo call | Sales | No login by Day 3 |

**Activation definition:** ≥1 lead OR ≥1 employee created within 7 days.

---

## Stage 2 — Activation (Week 1–2)

| Action | Owner |
|--------|-------|
| Confirm payment / plan selection | Sales |
| Handoff to CS (Enterprise) | Sales → CS |
| Kickoff call (30 min) — goals, team, modules | CS |
| Configure pipeline stages | CS guides admin |
| Import existing data (leads/employees) | CS assists |

---

## Stage 3 — Adoption (Month 1)

| Metric | Healthy | At Risk |
|--------|---------|---------|
| WAU / licensed users | > 50% | < 25% |
| CRM records created | > 20 leads or 5 deals | 0 |
| Modules used | ≥ 2 | 1 only |

**At-risk playbook:**
- Email: "Need help getting started?"
- Offer 1:1 training session
- Share relevant KB articles

---

## Stage 4 — Value (Month 2–3)

| Action | Description |
|--------|-------------|
| QBR (Enterprise) | Review metrics: pipeline, win rate, HR adoption |
| Usage report email (Growth) | Automated monthly summary |
| Case study request | If NPS > 8 |

---

## Stage 5 — Expansion (Month 3+)

| Signal | Action |
|--------|--------|
| Hitting user limit | Upsell seats |
| Asking about Finance/Support | Upsell to Enterprise |
| AI feature clicks | Offer AI add-on |
| Client portal inquiry | Client Portal add-on |

---

## Stage 6 — Renewal (Month 11)

| Action | Timing |
|--------|--------|
| Renewal reminder | 30 days before |
| Annual upgrade offer (20% savings) | 30 days before |
| Exit survey if churning | On cancellation |

---

## Health Score (Internal)

| Factor | Weight |
|--------|--------|
| Login frequency | 25% |
| Records created (30d) | 25% |
| Modules active | 20% |
| Support ticket sentiment | 15% |
| Payment status | 15% |

Score < 40 = red (CS outreach within 48h)

---

## CRM-Specific Success Milestones

- [ ] First lead imported
- [ ] First deal in pipeline
- [ ] First quote sent
- [ ] First invoice paid
- [ ] Sales dashboard reviewed by manager
- [ ] Full team onboarded
