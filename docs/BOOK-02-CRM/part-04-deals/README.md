## Purpose

Deals are the **revenue engine** of Vastora CRM. This part defines pipeline configuration, Kanban board, list views, deal detail, forecasting, and win/loss workflows — all built on Part 3's `contact_id` + `company_id` foundation.

**UI Reference:** Information density and pipeline workflow patterns inspired by [Zoho CRM](https://www.zoho.com/crm/) — **not copied**. Vastora identity: navy sidebar, `#296CB2` primary, Inter typography, 12px radius, canvas `#F8FBFF`.
